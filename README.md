# Gaming Reviews App

> [!NOTE]
> **Work in Progress**: This project is currently under active development. Features and UI are subject to change.

A modern web application designed to aggregate video game ratings from various sources. The core philosophy is to give users full control over which reviews and ratings they see, creating a personalized and transparent gaming discovery experience.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Data Source**: [IGDB API](https://www.igdb.com/api)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to use the app.
